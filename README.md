# Tour Ops Dashboard - Redeploy Version

This version is structured for Vercel and has the app folder at the project root.

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Deploy on Vercel

Upload this folder or push it to GitHub.
The project root should be this folder itself, since it contains:
- package.json
- app/

If Vercel asks for a Root Directory, leave it blank.
